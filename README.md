# SauceDemo Automation Framework

## Tech Stack
- Python
- Selenium WebDriver
- PyTest
- Page Object Model (POM)
- WebDriver Manager

## Project Structure
- tests/
- pages/
- utilities/
- conftest.py

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run tests:
   python -m pytest

## Features
- Login automation
- Add to cart
- Checkout flow
- End-to-end purchase validation
